NAME:

pwm

DESCRIPTION:

pwm is a secure password manager for the commandline

COMMANDS:

get                  Retrieves the value for KEY and prints it to STDOUT.       
help                 Display global or [command] help documentation.        
init                 Initializes a new store        
put                  Stores VALUE under KEY

GLOBAL OPTIONS:

-V, --verbose 
    Enable verbose output

-f, --file FILE 
    Determine the file that holds the store

-h, --help 
    Display help documentation

-v, --version 
    Display version information

-t, --trace 
    Display backtrace when an error occurs
